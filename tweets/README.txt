# Comment utiliser votre archive Twitter

La manière la plus simple de vous servir de votre archive Twitter passe par l'interface de navigation de l'archive fournie dans ce fichier. Double-cliquez simplement sur `index.html` dans le dossier racine et vous pourrez parcourir l'historique complet des Tweets dans votre navigateur.

---

Dans le dossier `data`, votre archive Twitter est présente sous deux formats : exports en JSON et CSV par mois et par année.

* Le CSV est un format générique qui peut être importé dans de nombreux outils de données et de tableurs ou qui peut être aisément utilisé à l'aide d'un langage de programmation.

## JSON pour les Développeurs

* L'export JSON contient une représentation complète de vos Tweets tels qu'ils sont retournés par la version 1.1 de l'API Twitter. Consultez https://dev.twitter.com/docs/api/1.1 pour plus d'informations.
* L'export JSON est également utilisé pour alimenter l'interface de navigation de l'archive (index.html).
* Pour utiliser l'export dans un analyseur générique JSON sous n'importe quel langage, supprimez la première et la dernière ligne de chaque fichier.
