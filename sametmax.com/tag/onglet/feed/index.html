Sous Mac (ça doit pas être bien différent sous Ubuntu et autre OS):

Aller à l’onglet de droite:
Alt + CMD + →

Aller à l’onglet de gauche:
Alt + CMD + ←

Aller à l’onglet le plus récemment utilisé:
Ctrl + TAB

Et inversement:
Ctrl + TAB + SHIFT

A celà vous pouvez rajouter la liste de Sam sur les astuces Sublime Text indispensables.
