Evidemment le fait de mettre un @login_required sur une classe, qui plus est si elle est étendue plus tard, fait tout planter. Du coup la solution que j’ai trouvé a été d’utiliser ça :

@method_decorator(login_required)

def dispatch(self, request, *args, **kwargs):

return super().dispatch(self, request, *args, **kwargs)
