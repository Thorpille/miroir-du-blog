Si vous êtes une grosse feignasse comme moi, faites un one liner: pmap $(ps aux | grep redis-server | grep -v grep | awk ‘{ print $2 }’)

Ou alors une petite fonction dans le .bashrc:

function get_memory() {
pmap $(ps aux | grep $1 | grep -v grep | awk ‘{ print $2 }’)
}
